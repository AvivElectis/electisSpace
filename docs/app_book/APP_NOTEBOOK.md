#### 1. Conference
- `infrastructure/conferenceStore.ts`: Persisted state for rooms.
- `application/useConferenceController.ts`: Logic for toggling meetings.

#### 2. Dashboard
- `presentation/DashboardPage.tsx`: Stats overview.
- `presentation/components/*`: Cards for Spaces, People, Sync status.

#### 3. People (Manager)
- `infrastructure/peopleStore.ts`: Persisted state for people.
- `application/usePeopleController.ts`: Import/Assign logic.
- `presentation/layouts/PeopleManagerLayout.tsx`: Specialized container.

#### 4. Settings
- `infrastructure/settingsStore.ts`: LocalStorage persistence for config.
- `presentation/SettingsDialog.tsx`: Tabbed configuration modal.

#### 5. Space
- `infrastructure/spacesStore.ts`: Core store for physical spaces.
- `application/useSpaceController.ts`: CRUD logic + Sync triggers.
- `presentation/SpacesPage.tsx`: Data grid for management.

#### 6. Sync
- `infrastructure/SolumSyncAdapter.ts`: AIMS API integration.
- `infrastructure/SFTPSyncAdapter.ts`: CSV/SFTP integration.
- `application/useSyncController.ts`: Background interval management.

#### 7. Update
- `application/useUpdateController.ts`: Electron IPC listeners.
- `presentation/UpdateNotification.ts`: Update prompt UI.

### Shared (`src/shared/`)

#### Infrastructure
- `services/solumService.ts`: Low-level API calls.
- `services/csvService.ts`: Parsing/Generating CSVs.
- `services/logger.ts`: Central logging system.
- `store/logsStore.ts`: Persisted logs.

#### Presentation
- `layouts/MainLayout.tsx`: Application shell (Sidebar/AppBar).

## Database Implementation üíæ

The application uses **IndexedDB** (via `idb-keyval`) for large datasets and **LocalStorage** for preferences.

### Stores (Zustand + Persistence)
1.  **spaces-storage** (IndexedDB): List of all `Space` entities.
2.  **people-storage** (IndexedDB): List of `Person` and `Assignment` entities.
3.  **conference-storage** (IndexedDB): List of `ConferenceRoom` entities.
4.  **logs-storage** (IndexedDB): System logs history.
5.  **settings-storage** (LocalStorage): App configuration.
6.  **sync-storage** (Session/LocalStorage): Connection status (ephemeral).
